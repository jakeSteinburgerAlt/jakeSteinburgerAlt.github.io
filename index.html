SpecOS is an operating system project from scratch, written in pure C and assembly.



Is this a Linux distro? No! SpecOS uses it's very own monolithic kernel, written completely from scratch.

Specs, please? FAT32 file system, 32 bit x86 architecture only, supports ATA hard disks, uses a monolithic kernel with a bitmap allocator. More info soon.

Current status? SpecOS is currently mostly a bunch of drivers mushed into a kernel, without a userspace yet. I've written drivers mostly for IO, time/date and a file system.

What's special about it? Literally nothing lol. This is just a project I'm doing to learn about kernel development, it's not really going to be... good.

Before you ask, yes, the background text is a real snippet of code from SpecOS.
